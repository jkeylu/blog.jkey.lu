python 中 *args 和 **kwargs

python 中的 args 和 *kwargs 真是个神奇的东西，看了书研究了一下 python 的参数匹配模型，顺便记录下来，感觉自己记性总是不怎么好。

1. 位置：从做至右进行匹配

3. 默认参数：为没有传入值的参数定义参数值

顾名思义，就是在定义函数时同时定义某个参数的默认值

4. 可变参数：收集任意多关于位置或关键字的参数
